# C++ Variable Name Validator Game

A game-based learning application built with Construct 3 for Lab Assignment 3.

## Concept
A "Flappy Bird" style game where players must fly through gates labeled with **Valid C++ Variable Names** and avoid **Invalid** ones.

## Learning Content
- **Valid Identifiers**: e.g., `myVar`, `count`, `_init`
- **Invalid Identifiers**: e.g., `1stPlace`, `class` (reserved), `my-var` (hyphens)

## Tech Stack
- Engine: Construct 3
- Platform: HTML5 (Web)
